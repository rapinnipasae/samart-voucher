ตั้งค่า Google Apps Script

วาง URL ที่ได้จากการ Deploy Google Apps Script เป็น Web App

วิธีตั้งค่า Google Apps Script

ขั้นตอน:
1. ไปที่ script.google.com และสร้างโปรเจกต์ใหม่
2. คัดลอก code จากไฟล์ google-apps-script.js ที่ได้รับมาวางใน editor
3. กด Deploy → New deployment → Web App
4. ตั้ง Execute as: Me | Who has access: Anyone
5. คัดลอก URL มาวางในช่องด้านบน
6. ระบบจะสร้าง Google Sheet และ Sheet ย่อยให้อัตโนมัติ
